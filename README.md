# 项目名称
论文复现
## Python 版本
该项目支持 Python 3.10。

## RICO 数据集
[RICO 数据集](http://www.interactionmining.org/rico.html) 

## 使用方法

1. 创建 Conda 环境：
   ```bash
   conda create -n your_env_name python=3.10
   ```

2. 激活环境：
   ```bash
   conda activate your_env_name
   ```

3. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```
## 启动方法
启动项目：
```bash
python VAEP.py
```
## 依赖项

- 请查看 `requirements.txt` 文件，以获取所有所需的库和版本。
